# Kalkulator Budżetu Domowego (Cała Rodzina)

Projekt aplikacji służący do łatwego i przejrzystego zarządzania budżetem domowym dla całej rodziny.

## 🛠 Technologie

*   **Język operacyjny:** PHP
*   **Baza danych:** Lokalna, działająca lokalnie i generowana przez kod aplikacji (bez potrzeby zewnętrznych serwerów bazodanowych).

## 📝 O projekcie

Aplikacja pozwala na bieżące kontrolowanie przychodów i wydatków wszystkich członków rodziny. Dzięki temu, że baza danych jest generowana lokalnie z poziomu kodu, projekt jest wyjątkowo łatwy w uruchomieniu na dowolnym środowisku obsługującym PHP.

## 🚀 Jak uruchomić?

1. Umieść pliki projektu w folderze serwera lokalnego (np. `htdocs` w XAMPP) lub użyj wbudowanego serwera PHP.
   *   Aby uruchomić wbudowany serwer PHP, otwórz terminal w folderze projektu i wpisz: `php -S localhost:8000`
2. Otwórz przeglądarkę i wejdź pod adres, pod którym uruchomiony jest projekt (np. `http://localhost:8000`).
3. Baza danych zostanie zainicjowana automatycznie przez skrypt PHP podczas pierwszego działania aplikacji.

## 📌 Funkcjonalności

*   Dodawanie i edycja wydatków oraz przychodów.
*   Zarządzanie kontami/profilami poszczególnych członków rodziny.
*   Podsumowania i analizy finansowe.
*   Lokalna baza "out of the box".
